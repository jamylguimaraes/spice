Welcom to Address Book TestCase-1
=================================

This TestCase1 is developed followed by BWE_Test-Case_1.pdf description. 


CONFIGURATION
=============

To process user need to follow these steps: 

1. Go to system/config file. You need to set base url, database host, database name, user and password. 
2. Import addressbook.sql file into the database using phpmyadmin or any other MySQL database tool.
